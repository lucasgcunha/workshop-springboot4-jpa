# Java Course Project

# Sobre o projeto

Aplicação criada para estudo de ferramentas como Java, Spring Boot, JPA / Hibernate e Maven.

# Tecnologias utilizadas
- Java
- Spring Boot
- JPA / Hibernate
- Maven

# Como executar o projeto
Pré-requisitos: Java 25

```bash
# clonar repositório
git clone https://github.com/lucasgcunha/workshop-springboot4-jpa

# executar o projeto
./mvnw spring-boot:run
```
